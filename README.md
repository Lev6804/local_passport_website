# LOCAL PASSPORT WEBSITE

## 🧩 Description
Ứng dụng website xác thực người dùng bằng **Passport Local Strategy** (giao diện web).

---

## ⚙️ Run project
```bash
npm install
npm start
```

Mở trình duyệt tại:  
👉 http://localhost:3000
